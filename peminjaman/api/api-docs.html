🔐 Authentication Endpoints

POST /api/auth/register - Register user baru
POST /api/auth/login - Login dan dapat JWT token
GET /api/auth/me - Info user saat ini (requires auth)

🏢 Facilities Endpoints

GET /api/facilities - List semua fasilitas (public)
GET /api/facilities/:id - Detail fasilitas
POST /api/facilities - Tambah fasilitas (admin only)
PUT /api/facilities/:id - Update fasilitas (admin only)
DELETE /api/facilities/:id - Hapus fasilitas (admin only)

📝 Loans Endpoints

GET /api/loans - List peminjaman
GET /api/loans/:id - Detail peminjaman
POST /api/loans - Buat peminjaman (requires auth)
PATCH /api/loans/:id/status - Update status (admin/staff)
DELETE /api/loans/:id - Hapus peminjaman

💬 Comments Endpoints

GET /api/comments/:loanId - List komentar
POST /api/comments/:loanId - Tambah komentar + foto (requires auth)
PATCH /api/comments/:loanId/mark-returned - Tandai selesai (admin/staff)
DELETE /api/comments/:commentId - Hapus komentar

📊 Reports Endpoints

GET /api/reports/summary - Dashboard statistik (admin/staff)
GET /api/reports/user-stats - Statistik per user
GET /api/reports/facility-usage - Laporan penggunaan fasilitas
